Palette de couleur :

Aide :

- Dessinez avec un clic de votre souris dans les 2 cadres du haut (point par point).

- Un shift + clic permet de commencer un nouveau segment à partir d'un nouvel endroit.

- La flèche vers le bas vous permet de lancer l'animation entre les 2 cadres et l'affiche dans le cadre juste en dessous.

- Il faut attendre 5s afin de refaire la même animation dans le même sens.

- La palette de couleur vous permet de changer de couleur, un clic gauche sur la couleur suffit à changer la couleur des prochains traits.
